# Binance Futures Testnet Trading Bot

## 📌 Overview
This is a simple Python-based trading bot that places MARKET and LIMIT orders on Binance Futures Testnet (USDT-M).

The application is built with a clean modular structure, proper logging, and input validation.

---

## 🚀 Features
- Place MARKET orders
- Place LIMIT orders
- Supports BUY and SELL sides
- Input validation
- Logging of API responses and errors
- Modular code structure

---

## 🛠️ Project Structure

- Client.py # Binance API connection
- Orders.py # Order execution logic
- Validators.py # Input validation
- Logging_config.py
- Cli.py # CLI entry point
